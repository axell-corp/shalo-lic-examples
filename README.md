# shalo-lic-examples

SHALO LICENSING のサンプル

# 提供サンプルの一覧

- `python_binding`: Python からアセットファイルの復号を行うためのものです。
- `with_pyinstaller`: pyinstaller と併用する際に、ユーザコードを暗号化する方法のサンプルです。
